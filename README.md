# GitHub Training Portal

A beginner-friendly web application that displays training resources stored in Supabase.

---

## Features

- Responsive design
- Dynamic training cards
- Supabase integration
- Shared CSS
- Shared JavaScript
- Multi-page website
- Easy to maintain

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Supabase
- GitHub Pages
